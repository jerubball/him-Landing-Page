Explaination of parameters

Parameter Explaination Values Allowed modes
mode Sets display mode.
  • unix
    • Unix timestamp in seconds.
  • unix10
  • countdown
    • Count to referenced time.
  • countup
    • Count from referenced time.
  • simple
    • Date and time in number.
  • plain
    • Date and time in full text.
  • him (default)
    • HIM timestamp.
time Sets referenced time.
  • Single integer representing Unix timestamp.
  • String formatted using RFC 2822 or ISO 8601 standard.
 countdown
countup
unit Sets display unit.
  • Single positive integer representing milliseconds.
  • second, sec
  • minute, min
  • hour, hr (default)
  • day
  • week, wk
  • month, mo
    • 1/12th of Gregorian calendar year. (30.436 days)
  • year, yr
    • Average Gregorian calendar year. (365.24 days)
countdown
countup
span Sets duration.
  • Single integer representing Unix timestamp.
  • String formatted using RFC 2822 or ISO 8601 standard.
  • Any number followed by space and units from above.
    • e.g. 100 year
  • shorter
    • 654,321 hours.
  • short
    • 765,432.1 hours.
  • long (default)
    • 876,543.21 hours.
  • longer
    • 987,654.321 hours.
countdown
base Sets base of number.
  • Any integer from 2 to 36.
  • binary, bin
    • base 2.
  • octal, oct
    • base 8.
  • decimal, dec (default)
    • base 10.
  • hexadecimal, hex
    • base 16.
unix
unix10
digit Sets decimal digit.
  • Any positive integer.
  • Default value depends on unit, between 3 and 6.
 countup
countdown
space Sets space character.
  • Any string.
  • \n
    • new line.
  • Default value is empty string.
 him
line Sets line character.
  • Any string.
  • \n (default)
    • new line.
simple
plain
